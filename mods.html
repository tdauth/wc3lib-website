The Power of Fire

The Power of Fire (German Die Macht des Feuers) is a Warcraft III: The Frozen Throne multiplayer RPG mod which allows up to six human players the share a co-op adventure in a fantasy world. It is realized by providing a custom .exe file which loads a custom MPQ archive. All participants need to download this .exe file and share the same MPQ archive. The map itself is relativly small like any other Warcraft III map.

Modification

Installation Setup

Download the installation setup and run it. Select your Warcraft III: The Frozen Throne directory as installation destination directory. The setup installs "The Power of Fire.exe" into your Frozen Throne directory and adds a symbolic link to your desktop. If you run "The Power of Fire.exe" you can start a custom game and select one of the maps in the subfolder "The Power of Fire".

Problems with Windows 8 and 10

The modification might have problems on Windows 8 or Windows 10. The game might freeze on starting "The Power of Fire.exe". One possible solution is to start "regedit.exe" and to go the the path "HKEY_CURRENT_USER\Software\Blizzard Entertainment\Warcraft III" and add the Registry entry "Allow Local Files" of type DWORD (32 Bit) with the value 1. Warcraft III loads all local files of the Warcraft III directory now. Therefore you have the extract the files of the MPQ archive "The Power of Fire.exe" into the Warcraft III directory. Use a tool like this. Check this out for further information.

Patch Error

If your Warcraft III has not been installed properly the registry entries might be missing. In this case the error "War3Patch not found" might appear when you start "The Power of Fire.exe". Use this tool to fix your registry entries and start "The Power of Fire.exe" afterwards. It should work now.

Resources

For getting all the resources download the latest Development KIT. It contains the MPQ archive "TPoF.mpq" which contains all models and textures etc. as well as the source code of the modification, the ASL and all the maps and campaigns.
